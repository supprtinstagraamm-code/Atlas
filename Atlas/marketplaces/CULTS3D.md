# Cults3D Playbook

**Status:** Experimental  
**Version:** 1.0.0  
**Owner:** Marketplace Lead  
**Review date:** 2026-10-07  
**Change history:** 1.0.0 — Initial Cults3D playbook.

## Role in Portfolio
Direct validation with buyers already seeking printable 3D files.
## Customer Intent
Assess geometry, print requirements, quality, license, and value before downloading.
## Suitable Products
Premium STL files with credible resin engineering and detailed technical presentation.
## Listing Requirements
Provide formats, parts, scale, dimensions, orientation, supports, hollowing, drainage, assembly, license, and verified settings.
## Image Sequence
Detailed hero render, orthographic views, assembly preview, cross-section, supported view, dimensions, and print photographs.
## Copy Structure
Begin with object function and differentiation, then present a concise technical specification.
## SEO Rules
Use accurate object, style, use, file, and printer terms observed in current relevant listings.
## Pricing and Licensing
Benchmark comparable file scope and presentation; offer commercial rights only with explicit terms.
## Success Evidence
Use visible engagement, reviews, downloads when shown, buyer questions, and support outcomes as labeled signals.
## Risks
Untested print claims, inadequate cross-sections, unclear license, and renders mistaken for prints.
## Release Checklist
Verify mesh, files, scale, technical images, tested status, print photos, description, license, and version.
